## Accessibility Features

### WCAG AAA Compliance
- 7:1 color contrast ratio minimum
- Semantic HTML structure
- Proper heading hierarchy (h1 > h2 > h3)
- ARIA labels and landmarks
- Skip links for keyboard users
- High contrast focus indicators

### Elder-Friendly Design
- 18px base font size (larger than typical 16px)
- Large touch targets (44px minimum, 48-56px recommended)
- Clear visual hierarchy
- Generous spacing and padding
- User-controllable text size
- Reduced motion support

### Keyboard Navigation
- All interactive elements keyboard accessible
- Skip links to main sections
- Visible focus indicators
- Proper tab order
- Arrow key support in carousel
